# Subsystem SDL2

`subsdl` is a lightweight subsystem of the SDL2 library, designed to simplify the process of creating 2D applications and integrating with various programming languages, including the xBase family. The subsystem offers a set of simple functions that enable quick prototyping and deployment of SDL2-based solutions, making it an ideal tool for developing projects with minimal effort.
